# Demo Asset Approval Pack (Draft)

Status: draft (needs explicit approval before any public use)

Use this checklist when preparing any demo assets for:
- blog posts
- Nomavek website pages
- Instagram promotion
- client-facing samples

---

## 1) What Assets Are Included?

Fill this list before sharing publicly.

- [ ] Images (count: ____)
- [ ] Videos (count: ____)
- [ ] Screenshots (count: ____)
- [ ] Client logos / names (yes/no)
- [ ] Site location references (yes/no)
- [ ] Floor plans / drawings (yes/no)

---

## 2) Source + Rights (Must Be Clear)

For every asset, record:
- Source: (client / stock / self-shot / generated / other)
- Rights: (allowed / not allowed / unknown)
- Notes: (credit line, restrictions, expiry)

If any rights are “unknown”, treat it as **not approved**.

---

## 3) AI-Generated Content (Extra Approval Needed)

If any asset is AI-generated:
- [ ] Explicit approval received to use AI-generated visuals publicly
- [ ] No misleading “real photo” framing
- [ ] No client-identifiable context unless approved

---

## 4) Privacy + Sensitivity Check

- [ ] No private addresses or unit numbers shown
- [ ] No faces or personal data
- [ ] No confidential drawings or markups

---

## 5) Final Approval Summary (Copy/Paste)

Approval request:
- Asset set name: __________________
- Intended channel: (blog / IG / website / client deck)
- Date range: ______________________
- Notes / restrictions: ____________

Decision:
- [ ] Approved
- [ ] Needs edits
- [ ] Not approved
